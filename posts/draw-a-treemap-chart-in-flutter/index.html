How to draw a Treemap chart in Flutter

A few weeks ago, I’ve started to work on a plotting library in Flutter. The few available packages to draw plots are only rendering simple plots like a line plot, pie plot or a bar plot.
If you have used before Javascript or Python, you probably know plotly and matplotlib. Those are powerful libraries with which we can render complex charts, charts scientists can use.

So among all the plots I’m adding to my package, I would like to focus on one chart that I find elegant. It’s the treemap. This chart displays hierarchical data in the shape of rectangle. A rectangle has a portion of the canvas area. It’s useful to quickly identify the biggest elements of a dataset.

For instance, some softwares are using it to display the hard disk space usage.
